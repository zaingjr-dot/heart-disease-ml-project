# Heart Disease Prediction Project

This project uses machine learning to predict whether a patient has heart disease based on medical data.

## Tools Used
- Python
- pandas
- scikit-learn
- matplotlib

## Steps
- Data cleaning (handled missing values)
- Feature processing
- Train-test split
- Logistic Regression model
- Model evaluation (accuracy, confusion matrix)

## Results
The model achieved around 81% accuracy.

## Goal
This project demonstrates how AI can be applied to healthcare data for predictive analysis.
